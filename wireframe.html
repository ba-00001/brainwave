Interface Design Report

1. Theme and Metaphor Development

Theme:

The theme for BrainWave Diagnostics revolves around neuroscientific exploration and precision diagnostics, combining scientific rigor with user-friendly design. The platform is crafted to appeal to a diverse user base, including healthcare professionals, researchers, and concerned parents. Drawing inspiration from the complexity of brain functions, the theme simplifies advanced AI diagnostics into an accessible and intuitive experience. Soft blues and greens create a calming interface for parents, while darker, clinical tones like navy and gray instill a sense of reliability for professionals. This thoughtful color scheme and polished design foster trust and confidence in the platform’s capabilities.

Metaphor for Interaction:

The interaction metaphor is inspired by waves and pathways, symbolizing brainwave activity and the diagnostic journey. Navigation flows smoothly, representing the intuitive progression from data upload to insights. This wave-like interaction mirrors the platform’s emphasis on ADHD diagnostics, providing a seamless, scientifically-backed user experience.

2. Icons and Graphics

Icons:

  • Brain Scan Upload: A stylized brain icon marks the upload section, where users can submit fMRI, SPECT, or qEEG scans.
  • Reports: Icons such as graphs and brainwave patterns help users locate data visualizations and diagnostic summaries in the report section.
  • AI-Powered Diagnostics: An AI robot icon highlights areas where machine learning analysis occurs, emphasizing the platform’s advanced AI capabilities.
  • Secure Data Handling: A padlock icon appears on upload and report pages, reinforcing HIPAA-compliant and secure data handling practices.

Whiteboard Elements:

Whiteboard elements, such as placeholders and instructional overlays, are integrated into the interface to enhance usability. These elements provide real-time guidance, helping users understand the function of each feature and navigate the platform seamlessly.

3. Screen Layout (Wireframes and Sketches)

Wireframe Overview:

The BrainWave Diagnostics interface prioritizes ease of use and clean navigation. The updated wireframes highlight the following key screens:

  • Welcome Screen: The landing page greets users with the tagline “Unlocking Insights, Empowering Decisions” and a “Start” button to guide them to the sign-up screen.
  • Sign-Up Screen: Users can create an account by entering their name, email, password, and role (Parent, Healthcare Professional).
  • Dashboard: Upon logging in, users land on a central dashboard where they can upload brain scans, generate new reports, or review past reports. Each section is clearly labeled and easily accessible.
  • Brain Scans Upload Screen: This screen offers options to upload specific brain scan types (fMRI, SPECT, qEEG) with identifiable icons for each type. A "New Report" button is also provided to initiate an AI-powered diagnostic report.
  • Report Highlights: This screen presents key findings from a report, offering users a quick overview before accessing more detailed information.
  • Detailed Report (fMRI): Users can view an in-depth analysis of an fMRI scan, with content tailored to the user’s role. Parents receive a simplified report with visual aids, while healthcare professionals see detailed metrics such as dopamine transporter levels and connectivity maps.
  • Previous Reports: Users can access a list of past reports, with options to view, download, or share each report. This feature enhances accessibility to historical data, supporting trend analysis.
  • Report Analysis Progress: This screen shows the status of ongoing analyses, with options to start a new analysis or monitor current progress.

Wireframe Sketches:

Each wireframe includes key navigation elements, with the exception of the Welcome Screen, Sign-Up Screen, and Report Analysis Progress Screen.

  • Header/Navigation Bar: Links to "Menu," "Notifications," "Share," and "Search" are consistently available across applicable screens, enabling easy access to main features and tools.

4. Media Content Production

Video Elements:

A short (2-3 minute) walkthrough video is located at the top section of the Dashboard. This video provides a step-by-step guide to using the platform, from uploading scans to generating and viewing reports. It enhances user onboarding and ensures a smooth introduction to the platform for new users.

3D Brain Visualization:

For healthcare professionals, an interactive 3D brain model is available on the Report Analysis Progress screen, allowing users to explore different brain regions relevant to ADHD diagnostics. This visualization enables rotation and zooming, providing an in-depth look at specific brain areas involved in the diagnostic process.

Audio Guides:

For parents, optional audio guides accompany diagnostic results, offering explanations in simple, accessible language. The option to activate these guides is located within the Header/Navigation Bar under the "Menu" in the hamburger icon, making it easy to turn on audio support as needed. These guides simplify complex information, making the diagnostic insights more understandable for non-professionals.
